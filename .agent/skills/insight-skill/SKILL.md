# 感悟系统 (Insight System)

感悟系统是角色的构建核心，呈六边形排布，相互影响且相生相克。

## 1. 六大感悟流派

### 动 (Dynamic)
- **核心**: 速度、连击、精力（AP）回复。
- **代表天赋**: 【动如脱兔】（每回合积攒连击率）、【雷厉风行】（轻功加成伤害）。

### 静 (Static)
- **核心**: 反击、等待、守门员。
- **代表天赋**: 【静以待变】（积攒反击率）、【黄雀在后】（协助队友反击）。

### 刚 (Hard)
- **核心**: 暴击、爆发、单体破坏。
- **代表天赋**: 【气吞山河】（孤军奋战加成）、【奋英雄怒】（低血量暴击加成）。

### 柔 (Soft)
- **核心**: 卸力、化劲、反弹伤害。
- **代表天赋**: 【四两拨千斤】（反转伤害）、【生生不息】（正面状态延长）。

### 巧 (Skill)
- **核心**: Debuff、侧背位打击、招式变化。
- **代表天赋**: 【指东打西】（使敌人转向）、【百花齐放】（多样招式加成）。

### 拙 (Simple)
- **核心**: 属性碾压、免疫控制、根基稳固。
- **代表天赋**: 【反璞归真】（免疫连击暴击反击）、【以拙胜巧】（大幅提升体质内功）。

## 2. 逻辑实现

在战斗生成过程中：
1. **触发检测**: 检测玩家当前激活的天赋（如：是否移动后攻击以触发【宁一思进】）。
2. **文本生成**: 根据带入的属性，输出符合流派的描写（如：“动”系描写侧重“残影”，“刚”系描写侧重“山崩”）。
3. **数值结算**: 将感悟加成应用于伤害、暴击、连击等判定。
