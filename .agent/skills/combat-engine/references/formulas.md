# 数值演算公式 (Combat Formulas)

## 核心战斗循环
1. **精力系统 (Action Points/AP)**: 每回合恢复固定精力（受内功与感悟影响）。移动、攻击、使用道具均消耗精力。
2. **攻击方向**:
   - 侧击：伤害 * 1.2
   - 背击：伤害 * 1.5
3. **内外功区分**:
   - **外功 (External)**: 伤害受“力道”与武器攻击力加成。
   - **内功 (Internal)**: 伤害受“内功”属性加成，无视部分护甲。

## 伤害计算公式
`Final_Damage = (Base_Skill_Damage + Weapon_Attack) * (1 + Primary_Attr/100) * Direction_Multiplier * Crit_Multiplier * Deflect_Reduction`

- **暴击 (Crit)**: 触发率受“刚”感悟影响，初始倍率 1.5。
- **反击 (Counter)**: 触发率受“静”感悟影响，反击招式消耗精力减半。
- **连击 (Combo)**: 触发率受“动”感悟影响，触发后恢复少量精力。

## 异常状态 (Debuffs)
- **内伤**: 降低内功输出，每回合扣除气血。
- **流血**: 移动时扣除气血，受力道属性加成。
- **中毒**: 降低全属性，持续数回合。
- **眩晕**: 无法行动，闪避/反击归零。
- **错骨**: 降低外功输出与连击率。

## 资质对修炼的影响
- **高资质 (>=80)**: 武学经验获取 +50%。
- **低资质 (<40)**: 习得特定功法（如《左右互搏》），连击判定次数 +1。
