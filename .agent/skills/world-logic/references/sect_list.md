# 江湖门派势力档案 (Sect & Faction Registry)
> *融合《逸剑风云决》与《河洛群侠传》的宏大江湖势力图谱*

---

## 势力关系总览

### 阵营划分
| 阵营 | 代表势力 | 核心理念 |
|------|----------|----------|
| 正道 | 武当、少林、丐帮、名剑山庄、赋闲书院 | 侠义为先，匡扶正义 |
| 中立 | 神鹰门、马帮、采玉帮、俏梦阁 | 利益优先，不涉纷争 |
| 邪道 | 五仙教、西域魔教、长生殿 | 各有所图，不拘常规 |
| 隐世 | 圣堂、先民部落、雪山派 | 远离尘世，独善其身 |

## 一、正道门派 (Orthodox Sects)

### 1. 武当派 (Wudang Sect)
- **宗旨**：以柔克刚，道法自然，济世度人
- **所在区域**：武当山区域
- **核心武学**：
  - 内功：纯阳无极功、太极心法
  - 武功：太极拳、太极剑、真武剑法
- **代表人物**：张君宝（青天明月）、王喆（侠之大者）
- **势力关系**：
  - 友好：少林寺（同为武林泰斗）、赋闲书院（儒道相通）
  - 敌对：西域魔教（正邪不两立）
- **特色**：道家圣地，太极发源之地，以内家功夫闻名天下

### 2. 少林寺 (Shaolin Temple)
- **宗旨**：禅武双修，普度众生，以武入禅
- **所在区域**：少林寺区域
- **核心武学**：
  - 内功：易筋经、洗髓经、金钟罩
  - 武功：罗汉拳、韦陀杵、少林七十二绝技
- **代表人物**：涤罪僧（佛光无涯）
- **势力关系**：
  - 友好：武当派（武林双璧）、丐帮（同御外侮）
  - 敌对：长生殿（佛魔不容）
- **特色**：天下武学之源，千年古刹，藏经阁武学浩如烟海

### 3. 丐帮 (Beggars' Sect)
- **宗旨**：行侠仗义，锄强扶弱，有饭同吃
- **所在区域**：丐帮总舵与分舵区域（遍布天下）
- **核心武学**：
  - 内功：逍遥游
  - 武功：打狗棒法、降龙十八掌、睡梦罗汉拳
- **代表人物**：李元兴（丐帮分舵主）
- **势力关系**：
  - 友好：少林寺、武当派（正道联盟）
  - 敌对：采玉帮（地盘冲突）
- **特色**：天下第一大帮，情报网络遍布江湖，帮众虽贫但义气深重

### 4. 名剑山庄 (Famous Sword Manor)
- **宗旨**：剑道至上，铸剑为魂，光耀门楣
- **所在区域**：名剑山庄区域（江南）
- **核心武学**：
  - 内功：名剑心法
  - 武功：名剑八式、烟雨剑法、名剑三十六路
- **代表人物**：欧阳雪（名剑千金）
- **势力关系**：
  - 友好：赋闲书院（江南世家）
  - 中立：大多数势力
- **特色**：江南剑派圣地，铸剑技艺天下无双，家风严谨

### 5. 赋闲书院 (Fuxian Academy)
- **宗旨**：文武并重，以德服人，教化天下
- **所在区域**：赋闲书院区域（江南）
- **核心武学**：
  - 内功：浩然正气心经、流星赶月
  - 武功：春秋笔法、婵娟一剑、醉八仙剑
- **代表人物**：颜普/东方迟（天下无双）、颜玉书（蕙语兰情）、文淬之（墨玉飞雪）
- **势力关系**：
  - 友好：武当派（儒道相通）、名剑山庄（江南世交）
  - 敌对：无（以和为贵）
- **特色**：儒门圣地，学子云集，以文会友亦文亦武

### 6. 点苍派 (Diancang Sect)
- **宗旨**：剑走轻灵，飘逸出尘，侠义为怀
- **所在区域**：姑苏与江南水乡区域（分支）
- **核心武学**：
  - 内功：点苍心法
  - 武功：点苍剑法、无双剑舞
- **代表人物**：漆笑儿（白马仙子）
- **势力关系**：
  - 友好：名剑山庄（剑道同源）
  - 中立：大多数势力
- **特色**：剑法飘逸，轻功卓绝，门下多才女

### 7. 重阳宫/全真教 (Chongyang Palace)
- **宗旨**：抗金复宋，济世救民，修道成仙
- **所在区域**：武当山区域（与武当相邻）
- **核心武学**：
  - 内功：先天功
  - 武功：全真剑法、七星七绝剑
- **代表人物**：王喆（侠之大者）
- **势力关系**：
  - 友好：武当派（道门同宗）
  - 敌对：金国势力
- **特色**：王重阳创立，心系苍生，具有强烈的民族情怀

## 二、中立势力 (Neutral Factions)

### 8. 神鹰门 (Divine Eagle Sect)
- **宗旨**：守护边疆，鹰击长空，维护秩序
- **所在区域**：天水城与神鹰门区域
- **核心武学**：
  - 内功：圆月神功、鹰扬神功
  - 武功：鹰爪功、神鹰九夺、神箭八法
- **代表人物**：冷鹰（神鹰门女捕头）、阿得阿克（神鹰再起）
- **势力关系**：
  - 友好：雪翼雕部落（盟友）
  - 中立：正道门派
  - 敌对：西域魔教（边境冲突）
- **特色**：西北重镇势力，擅长驯鹰，负责边境治安

### 9. 马帮 (Horse Caravan Guild)
- **宗旨**：货通天下，信誉为本，路见不平
- **所在区域**：茶马古道区域
- **核心武学**：
  - 内功：破阵曲
  - 武功：马家刀法、飞沙走石
- **代表人物**：禾郁青（马帮首领）
- **势力关系**：
  - 友好：各地商会
  - 中立：大多数势力
  - 敌对：山贼马匪
- **特色**：茶马古道霸主，运输网络发达，做得一手好菜

### 10. 采玉帮 (Jade Mining Guild)
- **宗旨**：采玉为业，利益至上，本土为王
- **所在区域**：漱玉矶区域
- **核心武学**：
  - 内功：霸王气劲
  - 武功：开山斧法、毒体傀儡
- **代表人物**：石鸿图（毒偶大师）
- **势力关系**：
  - 敌对：丐帮（地盘争夺）、外来势力
  - 中立：其他
- **特色**：漱玉矶地头蛇，控制玉石交易，作风霸道

### 11. 俏梦阁/红袖招 (Qiaomeng Pavilion / Red Sleeve Society)
- **宗旨**：情报为王，八面玲珑，庇护弱女
- **所在区域**：俏梦阁与平康区域
- **核心武学**：
  - 内功：迷魂大法、漫天花雨
  - 武功：化功大法、袖里乾坤、短兵相接
- **代表人物**：董娇儿（俏梦阁主）、樊香蝶（绮梦幽影）
- **势力关系**：
  - 友好：各方势力（情报交易）
  - 中立：不涉门派纷争
- **特色**：表面风月场所，实为江湖最大情报网络，庇护受苦女性

### 12. 淘石帮 (Stone Seekers Guild)
- **宗旨**：寻宝探秘，发财致富
- **所在区域**：漱玉矶区域（与采玉帮竞争）
- **核心武学**：
  - 内功：铁砂掌心法
  - 武功：铁砂掌、探囊取物
- **代表人物**：（普通帮众）
- **势力关系**：
  - 敌对：采玉帮（竞争对手）
- **特色**：专门赌石寻宝的小帮派

### 13. 江南武林联盟 (Jiangnan Martial Alliance)
- **宗旨**：联合自保，共御外侮
- **所在区域**：姑苏与江南水乡区域
- **核心武学**：各派武学汇集
- **代表人物**：吕仙儿（鸳鸯霜刀）、骆元玉（落地鹏）
- **势力关系**：
  - 友好：名剑山庄、赋闲书院
  - 中立：大多数势力
- **特色**：江南中小门派联合组织

## 三、邪道势力 (Unorthodox Factions)

### 14. 五仙教 (Five Immortals Sect)
- **宗旨**：巫蛊传承，守护苗疆，以毒制毒
- **所在区域**：十万大山与五仙教区域
- **核心武学**：
  - 内功：五仙秘术
  - 武功：千劫万毒手、虫笛音攻、蛊术
- **代表人物**：娜乌（五仙教圣女）
- **势力关系**：
  - 友好：五毒教（同源分支）、先民部落
  - 敌对：中原正道（误解）
- **特色**：苗疆圣地，用毒高手，实际心地善良只是不通中原礼教

### 15. 五毒教 (Five Venoms Sect)
- **宗旨**：以毒入道，天下无毒不丈夫
- **所在区域**：十万大山与五仙教区域
- **核心武学**：
  - 内功：五毒神功
  - 武功：绝毒秘典、万毒钻心
- **代表人物**：何白芍/何紫菀（比翼鸩羽）
- **势力关系**：
  - 友好：五仙教（同宗分支）
  - 敌对：中原正道
- **特色**：五仙教分支，更偏重毒术攻击

### 16. 西域魔教/玄火教 (Western Devil Cult / Xuanhuo Sect)
- **宗旨**：圣火永燃，统御天下，异教征服
- **所在区域**：西域区域、桃花林与蜃楼海域（渗透）
- **核心武学**：
  - 内功：圣火神功、玄火令
  - 武功：玄火掌、圣火燎原
- **代表人物**：瑶姬（玄火教圣女）
- **势力关系**：
  - 敌对：中原各大门派、神鹰门
- **特色**：西域神秘教派，正在向中原渗透，手段诡异

### 17. 长生殿 (Hall of Longevity)
- **宗旨**：追求长生，不择手段，永生不死
- **所在区域**：长生殿区域（隐世秘境）
- **核心武学**：
  - 内功：万毒心经
  - 武功：腐骨掌、追魂钉、炼丹术
- **代表人物**：荀杳杳（叛徒，绝命毒师）
- **势力关系**：
  - 敌对：少林寺（佛魔不容）、正道各派
- **特色**：追求长生的神秘组织，以活人炼丹闻名

### 18. 血刀门 (Blood Blade Sect)
- **宗旨**：以血祭刀，仇恨为力
- **所在区域**：江湖游历（无固定据点）
- **核心武学**：
  - 内功：血海魔功
  - 武功：血刀刀法
- **代表人物**：孔亮（复仇刀客）
- **势力关系**：
  - 敌对：所有门派
- **特色**：邪道散修，为复仇不择手段

### 19. 刺客组织/绝户门 (Assassin Guild)
- **宗旨**：收人钱财，替人消灾，使命必达
- **所在区域**：江湖游历（隐秘据点）
- **核心武学**：
  - 内功：绝户手
  - 武功：透骨钉、绝命十三剑
- **代表人物**：司徒荆（赶尽杀绝）
- **势力关系**：
  - 中立：只认钱不认人
- **特色**：江湖杀手组织，唯利是图但有自己的规矩

## 四、隐世势力 (Reclusive Factions)

### 20. 圣堂 (The Sanctuary)
- **宗旨**：守护远古秘密，传承天外意志
- **所在区域**：圣堂石窟
- **核心武学**：
  - 内功：霹雳心法
  - 武功：霹雳刀法、河洛四散
- **代表人物**：孔金（东豪，圣堂守护者）
- **势力关系**：
  - 中立：不涉世俗纷争
- **特色**：远古神秘组织，守护河图洛书碎片，传承天外武学

### 21. 雪山派 (Snow Mountain Sect)
- **宗旨**：冰心玉洁，与世隔绝，剑道修行
- **所在区域**：雪山派区域
- **核心武学**：
  - 内功：寒冰真气
  - 武功：胡家刀法、雪山剑法
- **代表人物**：容艳雪（冰肌玉骨，弃徒）
- **势力关系**：
  - 中立：极少与外界交流
- **特色**：终年积雪的隐世门派，以寒冰武学著称

### 22. 先民部落 (Ancient Tribe)
- **宗旨**：守护血脉，延续传承，与自然共生
- **所在区域**：十万大山与五仙教区域（深处）
- **核心武学**：
  - 内功：先民血脉
  - 武功：广寒剑法、九阴白骨爪（残）
- **代表人物**：颛孙凝（镜花水月）
- **势力关系**：
  - 友好：五仙教
  - 中立：不与外界接触
- **特色**：古老部族后裔，拥有神秘血脉力量，可召唤灵兽

### 23. 雪翼雕部落 (Snow Eagle Tribe)
- **宗旨**：与鹰共生，守护草原，勇猛善战
- **所在区域**：天水城与神鹰门区域（草原）
- **核心武学**：
  - 内功：鹰扬神功
  - 武功：神箭八法、鹰狂啄
- **代表人物**：阿得阿克（神鹰再起）
- **势力关系**：
  - 友好：神鹰门（盟友）
- **特色**：游牧民族，驯养雪翼雕，箭术超群

### 24. 隐剑谷 (Hidden Sword Valley)
- **宗旨**：剑道至尊，隐世修行，寻找传人
- **所在区域**：江湖游历（秘密地点）
- **核心武学**：
  - 内功：万剑归宗
  - 武功：天外飞仙、神门十三剑
- **代表人物**：江吟风（武林神话）
- **势力关系**：
  - 中立：超然物外
- **特色**：当世剑圣隐居之地，寻找可传承衣钵之人

### 25. 白帝剑派 (White Emperor Sword Sect)
- **宗旨**：剑道传承，光复门楣
- **所在区域**：中原与西南（游历）
- **核心武学**：
  - 内功：白帝心经
  - 武功：白帝剑法
- **代表人物**：上官虹（白帝传人）
- **势力关系**：
  - 友好：正道各派
- **特色**：没落剑派，传人游历江湖寻求复兴

## 五、特殊势力 (Special Factions)

### 26. 医圣谷 (Valley of the Divine Physician)
- **宗旨**：悬壶济世，救死扶伤，医者仁心
- **所在区域**：江湖游历（隐秘地点）
- **核心武学**：
  - 内功：药王神篇
  - 武功：金针渡劫、万物回春、子午针灸经
- **代表人物**：程雁华（藏花妙手）
- **势力关系**：
  - 友好：所有门派（医者无敌）
- **特色**：以医术闻名，不问正邪只救病人

### 27. 盗神门 (Divine Thief Sect)
- **宗旨**：盗亦有道，劫富济贫
- **所在区域**：桃花林与蜃楼海域区域（漱玉矶）
- **核心武学**：
  - 内功：侠盗心法
  - 武功：离合掌、盗神步

- **势力关系**：
  - 敌对：采玉帮（常有冲突）
  - 中立：其他门派
- **特色**：天下第一神偷门派，虽非正道却信守诺言

### 28. 修罗殿 (Asura Hall)

- **势力关系**：
  - 敌对：所有门派
