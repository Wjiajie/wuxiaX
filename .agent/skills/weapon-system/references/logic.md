# 武器系统逻辑参考 (Logic Reference)

## 1. 锻造逻辑 (Forging)
锻造成功率及属性加成计算：
`属性 = 材料基础值 * (1 + 铁匠倍率) * 随机浮动(0.9~1.1)`

## 2. 耐久度损耗 (Durability Consumption)
每次攻击或格挡时的损耗判定：
- **普通攻击**：损耗 1 点，5% 概率触发“意外受损”额外损耗 2 点。
- **猛烈格挡**：损耗 2-5 点，取决于对手攻击力。
- **神兵判定**：若武器品级为“神兵”，损耗速度降低 50%。

## 3. 耐久度对性能的影响
- `耐久 > 50%`：发挥 100% 威力。
- `20% < 耐久 <= 50%`：威力下降 20%。
- `耐久 <= 20%`：威力下降 50%，且有 1% 概率在战斗中直接断裂。
