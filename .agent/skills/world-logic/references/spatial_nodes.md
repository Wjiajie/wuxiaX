# 空间节点档案 (Spatial Nodes) - 资源与特点详解

## 1. 拱石村区域 (Gongshi Village)
- **特点**：新手福地，民风淳朴，地势平缓。
- **盛产资源**：
    1. 【药草】：后山及路边，用于制作基础疗伤药。
    2. 【铁砂】：河滩边采集，用于修炼铁砂掌。
    3. 【野猪肉】：猎杀野猪获得，补充体力的主要食材。
    4. 【生锈铁片】：民宅或废墟搜索，低级锻造材料。

## 2. 漱玉矶区域 (Shuyu Shoal)
- **特点**：湿润滩涂，石多金少，黑市出没。
- **盛产资源**：
    1. 【玉石原石】：采玉坊及河床，可赌石开出翡翠。
    2. 【翡翠/玛瑙】：高级玉石，极高交易价值。
    3. 【盐】：海边或商船，烹饪及任务物品。
    4. 【沙石】：建筑材料，偶尔用于暗器。

## 3. 大研镇区域 (Dayan Town)
- **特点**：极其繁华，建筑错落，势力盘根错节。
- **盛产资源**：
    1. 【精金/镔铁】：铁匠铺及宝库，高级武器核心材料。
    2. 【丝绸/锦缎】：布庄及木府，制作高级护甲。
    3. 【陈年佳酿】：酒楼及俏梦阁，用于交友及回气。
    4. 【各类书信】：通过偷窃或搜索获得，开启隐藏任务。
    5. 【古玩珍宝】：珍宝阁及富户家，换取大量名望或金钱。

## 4. 遂宁与丈人村区域 (Sunning)
- **特点**：丛林茂密，竹影横斜，猎人居所。
- **盛产资源**：
    1. 【翠竹】：竹林采集，用于制作长柄武器及陷阱。
    2. 【兽皮】：狩猎野兔、野猪获得，制作皮甲。
    3. 【山珍/蘑菇】：雨后林间，珍稀食材。
    4. 【金银花/甘草】：药田盛产，解毒良药。

## 5. 卧龙村与石家庄区域 (Wolong)
- **特点**：依山而建，梯田层叠，石堡坚固。
- **盛产资源**：
    1. 【稻米】：梯田特产，主食资源。
    2. 【寒铁】：后山深处矿脉，打造神兵必备。
    3. 【花种】：庄园花园，陶冶情操。
    4. 【石块】：随处可见，投掷类暗器基础。

## 6. 极地与秘境 (Forbidden Realms)
- **极北雪域**：
    - 特点：终年积雪，生灵罕至。
    - 资源：【天山雪莲】、【白熊胆】、【万年冰泉】。
    - 🔮 **河图洛书**：【洛书·天璇碎片】藏于冰峰之巅，需抗寒装备+轻功≥20
- **西南密林**：
    - 特点：瘴气弥漫，蛇虫鼠蚁。
    - 资源：【五毒（蝎子、蜈蚣等）】、【断肠草】、【沉香木】。
    - 🔮 **河图洛书**：【洛书·天玑碎片】藏于毒瘴深处，需持有【避毒珠】
- **月牙岛**：
    - 特点：孤悬海外，地火涌动。
    - 资源：【玄铁】、【赤火砂】、【深海贝壳】。
    - 🔮 **河图洛书**：【河图·离位残片】藏于火山口，需抗火装备或内功≥25

## 7. 圣堂石窟 (Sanctuary Cavern)
- **特点**：远古遗址，蕴含天外意志，传闻中的觉醒地之一。
- **盛产资源**：
    1. 【遗迹断碑】：记载远古武学残篇。
    2. 【入门歌诀】：圣堂传承基础心法。
    3. 【天外陨铁】：极其稀有，锻造神兵利器。
- **相关 NPC**：孔金（东豪，圣堂守护者）
- **相关势力**：圣堂（隐世组织）
- 🔮 **河图洛书**：【河图·乾位残片】藏于深处

## 8. 武当山区域 (Wudang Mountain)
- **特点**：道家圣地，云雾缭绕，清修福地。山势雄浑，道观林立，太极发源之地。
- **盛产资源**：
    1. 【灵芝/何首乌】：悬崖峭壁采集，延年益寿的珍贵药材。
    2. 【松针/道茶】：道观特产，清心凝神，恢复内力。
    3. 【玄铁精�ite】：后山矿脉，铸剑上品。
    4. 【丹砂/朱砂】：炼丹材料，亦可入药驱邪。
    5. 【道经残卷】：藏经阁及隐蔽处，习得道家武学。
- **相关 NPC**：张君宝（青天明月）、王喆（侠之大者）
- **相关势力**：武当派、重阳宫（全真教）

## 9. 少林寺区域 (Shaolin Temple)
- **特点**：佛门圣地，千年古刹。晨钟暮鼓，武僧如云，天下武学之源。
- **盛产资源**：
    1. 【舍利子】：高僧圆寂后遗留，极其珍贵。
    2. 【佛经/武学秘笈】：藏经阁珍藏，易筋经、洗髓经残本。
    3. 【铁砂/铜砂】：练功房，修炼金钟罩铁布衫。
    4. 【素斋食材】：寺院菜园，清心素食。
    5. 【禅杖/木鱼】：法器，亦可作武器使用。
- **相关 NPC**：涤罪僧（佛光无涯）
- **相关势力**：少林寺

## 10. 天水城与神鹰门区域 (Tianshui City & Divine Eagle Sect)
- **特点**：西北重镇，商贾云集，神鹰门总舵所在。城墙高耸，鹰隼盘旋，边塞风情浓郁。
- **盛产资源**：
    1. 【鹰羽/鹰爪】：神鹰门特产，制作暗器及护具。
    2. 【西域香料】：丝绸之路商品，烹饪及炼药。
    3. 【驼毛/羊皮】：牧民贸易品，制作保暖装备。
    4. 【铁器/马蹄铁】：铁匠铺，边塞武器锻造。
    5. 【枣/葡萄干】：特产干果，恢复体力。
- **相关 NPC**：冷鹰（神鹰门女捕头）、阿得阿克（神鹰再起）
- **相关势力**：神鹰门、雪翼雕部落

## 11. 名剑山庄区域 (Famous Sword Manor)
- **特点**：江南剑派圣地，依山傍水，烟雨朦胧。庄内剑阁林立，铸剑炉火不息。
- **盛产资源**：
    1. 【名剑/宝剑】：山庄铸剑，天下闻名。
    2. 【精铁/玄铁】：专属矿脉，铸剑必备。
    3. 【剑谱/剑诀】：剑阁收藏，名剑八式等绝学。
    4. 【丝绸/锦缎】：江南特产，制作轻甲。
    5. 【龙井茶】：庄园茶田，提神醒脑。
- **相关 NPC**：欧阳雪（名剑千金）
- **相关势力**：名剑山庄

## 12. 赋闲书院区域 (Fuxian Academy)
- **特点**：儒门圣地，书香门第。学子云集，以文会友，亦文亦武。
- **盛产资源**：
    1. 【古籍/经卷】：藏书楼珍藏，增长见识。
    2. 【笔墨纸砚】：文房四宝，制作符咒及书信。
    3. 【君子兰/梅花】：书院花园，陶冶性情。
    4. 【酒/茶】：文人雅集必备，恢复内力。
    5. 【琴谱/棋谱】：琴棋书画传承。
- **相关 NPC**：颜玉书（蕙语兰情）、颜普/东方迟（天下无双）、文淬之（墨玉飞雪）
- **相关势力**：赋闲书院

## 13. 姑苏与江南水乡区域 (Gusu & Jiangnan)
- **特点**：水乡泽国，小桥流水，园林遍布。丝竹声声，才子佳人汇聚之地。
- **盛产资源**：
    1. 【丝绸/刺绣】：苏绣闻名天下，制作高级护甲。
    2. 【莲子/荷叶】：池塘采集，清热解毒。
    3. 【鲜鱼/河蟹】：水产丰富，烹饪食材。
    4. 【太湖石】：园林点缀，亦可制作暗器。
    5. 【折扇/团扇】：江南特产，可作武器。
- **相关 NPC**：吕仙儿（鸳鸯霜刀）、戚如潇（追魂索命）、骆元玉（落地鹏）
- **相关势力**：点苍派（分支）、江南武林联盟

## 14. 十万大山与五仙教区域 (Hundred Thousand Mountains & Five Immortals Sect)
- **特点**：南疆秘境，瘴气弥漫，毒虫遍地。苗疆圣地，巫蛊之术盛行。
- **盛产资源**：
    1. 【五毒虫】：蝎子、蜈蚣、蛇、蟾蜍、蜘蛛，炼毒必备。
    2. 【断肠草/见血封喉】：剧毒植物，制毒材料。
    3. 【苗银/银饰】：苗族工艺，可驱邪避毒。
    4. 【蛊虫】：圣女专属，神秘力量载体。
    5. 【虫笛/蛊壶】：操控毒虫的法器。
- **相关 NPC**：娜乌（五仙教圣女）、何白芍/何紫菀（比翼鸩羽）、燕未还（南疆拳王）、荀杳杳（绝命毒师）、颛孙凝（镜花水月）
- **相关势力**：五仙教、五毒教、先民部落

## 15. 茶马古道区域 (Tea-Horse Road)
- **特点**：崇山峻岭，古道蜿蜒。马帮往来，连接中原与西南边陲。
- **盛产资源**：
    1. 【普洱茶/砖茶】：茶马交易主货，亦可入药。
    2. 【骡马】：运输工具，亦可作战马。
    3. 【火腿/腊肉】：马帮干粮，恢复体力。
    4. 【药材/草药】：沿途采集，各类珍稀药材。
    5. 【皮革/马鞍】：马帮装备，制作护具。
- **相关 NPC**：禾郁青（马帮首领）
- **相关势力**：马帮

## 16. 桃花林与蜃楼海域区域 (Peach Blossom Grove & Mirage Sea)
- **特点**：东海仙境，桃花漫天，海市蜃楼时隐时现。武家旧事的核心区域。
- **盛产资源**：
    1. 【桃花/桃木】：制作法器，辟邪驱魔。
    2. 【珍珠/珊瑚】：海底采集，极高价值。
    3. 【海盐/海藻】：海边特产，烹饪及炼药。
    4. 【贝壳/砗磲】：制作饰品及暗器。
    5. 【奇鱼/海参】：珍稀食材，滋补圣品。
- **相关 NPC**：步微月（碧海扇仙）、瑶姬（玄火教圣女）、叶飞（孤锋剑客）、李叹/舒光壁（盗神）、漆笑儿（白马仙子）、石鸿图（毒偶大师）
- **相关势力**：采玉帮、玄火教（渗透势力）、点苍派

## 17. 西域区域 (Western Regions)
- **特点**：大漠孤烟，黄沙漫天。丝路重镇，异域风情，魔教潜伏之地。
- **盛产资源**：
    1. 【天山雪莲】：雪山绝顶，延年益寿圣药。
    2. 【西域宝石】：红宝石、蓝宝石，价值连城。
    3. 【驼峰/驼绒】：沙漠生存必需品。
    4. 【葡萄酒/马奶酒】：西域特产，恢复内力。
    5. 【弯刀/飞镖】：西域兵器，锋利异常。
- **相关 NPC**：隼离（孤鹰决剑）
- **相关势力**：西域魔教、玄火教

## 18. 雪山派区域 (Snow Mountain Sect)
- **特点**：终年积雪，冰峰耸立。雪山派驻地，寒气逼人，非常人可至。
- **盛产资源**：
    1. 【天山雪莲】：悬崖绝壁，极难采集。
    2. 【万年寒冰】：炼制寒冰武器必备。
    3. 【雪狐皮/白貂皮】：极品毛皮，保暖圣品。
    4. 【冰晶/雪玉】：天然形成，制作冰属性武器。
    5. 【雪参/冬虫夏草】：滋补圣品，大幅恢复内力。
- **相关 NPC**：容艳雪（冰肌玉骨）
- **相关势力**：雪山派

## 19. 丐帮总舵与分舵区域 (Beggars' Sect)
- **特点**：遍布天下，行踪不定。丐帮弟子行走江湖，行侠仗义，消息灵通。
- **盛产资源**：
    1. 【狗肉/野味】：丐帮特色，恢复体力。
    2. 【烈酒/浊酒】：丐帮必备，增加豪气。
    3. 【打狗棒】：帮主信物，不可交易。
    4. 【破衣烂衫】：伪装道具，降低仇恨。
    5. 【江湖情报】：丐帮网络，消息灵通。
- **相关 NPC**：李元兴（丐帮分舵主）
- **相关势力**：丐帮

## 20. 俏梦阁与平康区域 (Qiaomeng Pavilion & Pingkang)
- **特点**：繁华烟柳之地，消息汇集中心。表面风月场所，实为江湖情报网络。
- **盛产资源**：
    1. 【胭脂水粉】：化妆品，亦可制作迷药。
    2. 【丝绸/轻纱】：制作轻便护甲。
    3. 【美酒佳酿】：各色名酒，恢复内力。
    4. 【琴瑟/箫笛】：乐器，亦可作武器。
    5. 【秘密书信】：情报交易品，开启隐藏任务。
- **相关 NPC**：董娇儿（俏梦阁主）、樊香蝶（绮梦幽影）
- **相关势力**：俏梦阁、红袖招

## 21. 长生殿区域 (Hall of Longevity)
- **特点**：隐世秘境，追求长生不老的神秘组织驻地。传闻中掌握延寿秘术。
- **盛产资源**：
    1. 【长生丹/不老药】：传说中的仙丹。
    2. 【奇珍异草】：各类延年益寿的珍稀药材。
    3. 【炼丹炉/丹方】：炼丹设备及秘方。
    4. 【符箓/法器】：道门法器，驱邪护身。
    5. 【古籍秘典】：长生秘术相关典籍。
- **相关 NPC**：荀杳杳（叛徒，现已出逃）
- **相关势力**：长生殿
