# 物品与锻造系统 (Items & Crafting)

## 1. 消耗品
- **智慧果**: 增加资质。
- **小还丹**: 恢复生命。
- **槟榔**: 基础恢复。

## 2. 材料与矿产
- **玄铁/寒铁/镔铁**: 用于打造传说武器。
- **紫颤木/皮毛**: 弓箭与特殊防具材料。

## 3. 传说武器锻造 (Legendary Forging)
必须在 **月牙岛铁匠** 处，持有【铁匠令】和相应蓝图进行打造。
- **玄铁剑**: 玄铁 x8。
- **后羿弓**: 长弓蓝图 + 雪豹筋 x1 + 紫颤木 x6。
- **金乌刀**: 朴刀蓝图 + 寒铁 x6。

## 4. 特殊神器：雷神之锤
- **获取链**: 漱玉矶墓地（见盗墓兄弟） -> 俏梦阁外盗墓兄弟 -> 大研镇墓穴调查（获赠诺基亚 3210） -> 月牙岛练无情对话抉择。
