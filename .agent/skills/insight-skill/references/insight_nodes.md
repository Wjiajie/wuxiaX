# 感悟系统 (Insight System) - 六脉感悟

## 动 (Motion) - 唯快不破
- **【动如脱兔】**: 每回合连击率+2%。触发连击回复 1 点精力。
- **【雷厉风行】**: 移动格数越多，伤害加成越高（每格+5%）。
- **【宁一思进】**: 攻击后可消耗剩余精力再次移动。
- **【先发制人】**: 战斗开始时，额外获得 2 点精力。

## 静 (Stillness) - 以静制动
- **【静以待变】**: 等待指令后，反击率与闪避率大幅提升。
- **【黄雀在后】**: 守御范围扩大。敌人进入攻击范围即触发截击。
- **【借力打力】**: 反击时，额外附加对方 40% 的攻击伤害。
- **【借尸还魂】**: 击杀敌人后，回复消耗的 50% 精力。

## 刚 (Hard) - 强攻无敌
- **【气吞山河】**: 暴击时，无视对方 30% 护甲。
- **【奋英雄怒】**: 血量低于 50% 时，暴击率翻倍。
- **【以刚制柔】**: 攻击无法被卸力，且对“柔”系敌人有额外伤害。

## 柔 (Soft) - 绵里藏针
- **【四两拨千斤】**: 受到攻击时，将伤害的 30% 反弹给周围敌人。
- **【生生不息】**: 每回合自动回复 5% 内力与气血。
- **【拨云见日】**: 卸力触发后，下次攻击必中且带有虚招效果。

## 巧 (Skill) - 变幻莫测
- **【指东打西】**: 攻击有概率使敌人眩晕或转向。
- **【百花齐放】**: 每回合使用不同的招式，伤害累积增加。
- **【暗箭难防】**: 远程或暗器攻击无视障碍物，且背击伤害+50%。

## 拙 (Simple) - 大巧若拙
- **【反璞归真】**: 免疫敌方连击与暴击。
- **【以拙胜巧】**: 体质对伤害的转化率提高。
- **【稳扎稳打】**: 站立不动时，防御力提升 50%。
