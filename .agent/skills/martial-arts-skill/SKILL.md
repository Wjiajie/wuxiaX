# 武学功法全书 (Martial Arts)

武学是文本江湖中提升实力的阶梯，分为内功与外功。

## 1. 内功心法 (Internal Arts)

内功决定内力（MP）上限、回气速度。
- **正气诀**: 入门赠送。
- **北冥秘籍**: 资质 >= 70 在书院习得。
- **洗髓经**: 破庙复杂剧情获得。
- **先天功**: 王喆主线 + 柔系全满。

## 2. 外功招式 (External Arts)

- **搏击/指法**: 六路剑指、段家指。
- **剑法**: 独孤九剑、段家剑。
- **刀法**: 金乌刀法。
- **奇门/箭法**: 飞雪神箭。

## 3. 修炼逻辑

1. **研读 (Study)**: 消耗经验值提升秘籍等级。
2. **根基判定**: 习得特定武功可能需要特定感悟（动/静/刚/柔等）或属性（资质/内功）。
