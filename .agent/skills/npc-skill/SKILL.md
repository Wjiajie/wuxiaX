# NPC 与生态系统 (NPC & Beastiary)

## 1. 动态关系网系统
> *江湖之中，恩怨情仇如蛛丝般交织。一言一行，皆可牵动人心向背。*

### 1.1 好感度机制
每个NPC维护一个 `-100 ~ +100` 的好感度数值，决定其对主角的态度：
- **死敌** (-100 ~ -75)：不共戴天，见面即战
- **仇视** (-74 ~ -50)：拒绝合作，可能设伏
- **冷淡** (-49 ~ -25)：对话生硬，不愿透露信息
- **陌生** (-24 ~ 24)：萍水相逢，中立态度
- **友善** (25 ~ 49)：愿意提供帮助与线索
- **信任** (50 ~ 74)：可招募入队，共享秘密
- **挚友** (75 ~ 100)：生死之交，解锁专属剧情与传承

### 1.2 人格偏好联动
NPC对主角的四维人格（正/邪/狂/狷）有不同偏好，影响好感度增减速率：
- 正派NPC（如段思平）对「正」值高者更易产生好感
- 邪道NPC（如何紫菀）对「邪」值高者态度更友善
- 隐士NPC（如扫地僧）偏好「狷」值高的内敛之人

## 2. 队友互动系统

### 2.1 招募机制
- **基础条件**：好感度达到「信任」(≥50)
- **专属条件**：部分队友需完成特定任务或满足人格阈值
- **队伍上限**：最多同时携带4名队友

### 2.2 羁绊效果
队友间存在特殊羁绊关系，触发联动增益：
- **【双剑合璧】**：李叹 + 漆笑儿同队，攻击+15%
- **【心有灵犀】**：颜玉书好感度达90，解锁隐藏结局线
- **【回春妙手】**：程雁华好感度达60，战后自动恢复气血

### 2.3 阵营冲突
正邪阵营队友同时在队时，可能触发内部矛盾剧情，需玩家调解。

## 3. 核心队友记录
- **孔金**: 新手导师，正气凛然，圣堂接引人。
- **段思平**: 段氏皇族，正道核心。
- **张君宝**: 柔系巅峰，少年宗师。
- **程雁华**: 核心治疗者，北方医者。

## 4. 兽王图鉴 (Beast Kings)
野外分布着强力的兽王，可击杀或通过【兽王册】捕捉。
- **白熊王**: 极北雪域，附带冻结。
- **雪豹王**: 西北雪山，高闪避。
- **白鳄鱼**: 死亡沼泽，需【纪玫湘任务】触发。
- **猿王 (猴三)**: 东部海岛，通过解谜开启。

## 5. 互动逻辑
- **招募**: 需完成特定主线或满足善恶值要求。
- **捕捉**: 使用 `use beast_manual` 指令，目标 HP < 20% 时成功率上升。
- **阵营**: 善恶值（正/邪）直接影响 NPC 的初始态度和交互分支。
