# 江湖律令与运行法则 (World Rules)

## 0. 天外遗迹：圣堂法则 (Sanctuary Rules)
- **圣堂石窟**：远古圣堂遗址，蕴含天外意志。
- **意志共鸣**：
    - 进入石窟时自动触发，读取四维人格(正/邪/狂/狷)进行共鸣判定
    - 共鸣类型 = 最高人格维度；共鸣强度 = 最高值 - 次高值
    - 详见 `story-engine/references/sanctuary_will.md`
- **圣堂赐福**：
    - 正念共鸣：【正气护体】内功+3
    - 邪念共鸣：【阴魂不散】夜间行动+15%
    - 狂念共鸣：【战意沸腾】首战攻击+20%
    - 狷念共鸣：【洞察秋毫】观察判定+25%
- **隐藏机制**：共鸣强度≥20时，显现【圣堂断碑】关键道具

## 1. 天时：十二时辰轮转
- **昼夜判定**：
    - **卯、辰、巳、午、未、申 (白天)**：市集繁荣，正道名门正常运作。
    - **酉、戌、亥、子、丑、寅 (黑夜)**：视距降低 50%，潜行成功率增加 25%。特定刺杀、盗宝任务仅在此开启。
- **气候影响**：
    - 雪域严寒：每移动一个节点消耗双倍精力，需饮用【烧酒】抵御。
    - 密林瘴气：需佩戴【避毒珠】或服用【解毒丹】，否则每回合损失气血。

## 2. 人心：人格与善恶 (Moral & Personality)
- **人格四维 (Personality Quads)**：
    - **狷狂 (Wild)** vs **淳和 (Gentle)**：狷狂者对话激进，易得颜玉书等奇才青睐；淳和者言语温良，易获长辈赏识。
    - **羁骜 (Rebellious)** vs **坦荡 (Upright)**：羁骜者行事不拘格套，适合修习暗黑武学；坦荡者重信守诺，正道名声增长极快。
- **善恶值 (Morality)**：
    - **大善 (Saint)**：购买物品享 8 折，丐帮、青城派等正道门派好感初始为【友善】。
    - **极恶 (Villain)**：遭遇正道人士追杀（触发随机战斗），但可修习【百花缭乱】等禁忌绝学。

## 3. 势：势力纠葛 (Faction Logic)
- **声望 (Prestige)**：
    - 区域声望：在特定区域（如大研镇）完成任务可提升，声望达“名动一时”可开启该地隐藏商铺。
    - 势力敌对：协助禾家马帮将导致与当地黑道势力敌对，进入其领地（如海盗营地）将自动触发战斗。

## 4. 命：奇遇与福缘 (Adventure & Luck)
- **福缘 (Luck)**：影响资源采集的产出倍率。
- **跳跃与轻功判定**：
    - 环境中高处、悬崖下的闪光点需 `jump up` 指令。
    - 判定逻辑：`if Protagonist.Lightness >= Node.Requirement: Success else: Fall(Damage)`。
- **奇遇触发**：需同时满足【时辰】+【地点】+【特定人格/善恶】。
    - *示例*：半夜子时，身怀大恶，在大研镇后巷搜索，可触发【刺客组织招揽】。
