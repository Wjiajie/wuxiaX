# 序章脚本：天外来客 (Prologue)

## 阶段一：觉醒
- **环境**: 圣堂石窟（远古遗迹，石壁刻有模糊符文，空气中弥漫着淡淡灵气）
- **前置条件**: 主角初入石窟，触发圣堂意志共鸣判定
- **共鸣机制**: 参见 [sanctuary_will.md](sanctuary_will.md) 进行四维人格共鸣判定
- **对话**: 孔金反应根据共鸣类型变化（正念→惊喜，邪念→试探，狂念→大笑，狷念→欣赏）
- **互动**:
  - 必得：【正气歌诀】（内功心法入门）
  - 赐福：根据共鸣类型获得对应被动技能
  - 隐藏：共鸣强度 ≥ 20 时，显现【圣堂断碑】
- **指令提示**: `study zhengqi_gejue`
- **状态更新**: 更新 character_sheet.md 添加圣堂赐福被动

## 阶段二：遭遇战
- **触发**: 山洞前行至 (X, Y, Z) 触发剧情。
- **对话**: 黑衣人拦路。
- **抉择**: 
  - A：“你们是什么人？” -> 直接进入战斗。
  - B：“有话好商量...” -> 影响善恶值或特定分支。

## 阶段三：战后
- **指令提示**: `loot corpse`。
- **物品获得**: 【夜行衣】x1。
- **成就**: 使用“野球拳-蓄劲”获胜，解锁【才不会放了你呢】。
