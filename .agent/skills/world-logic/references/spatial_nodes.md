# 空间节点档案 (Spatial Nodes) - 资源与特点详解

## 1. 拱石村区域 (Gongshi Village)
- **特点**：新手福地，民风淳朴，地势平缓。
- **盛产资源**：
    1. 【药草】：后山及路边，用于制作基础疗伤药。
    2. 【铁砂】：河滩边采集，用于修炼铁砂掌。
    3. 【野猪肉】：猎杀野猪获得，补充体力的主要食材。
    4. 【生锈铁片】：民宅或废墟搜索，低级锻造材料。

## 2. 漱玉矶区域 (Shuyu Shoal)
- **特点**：湿润滩涂，石多金少，黑市出没。
- **盛产资源**：
    1. 【玉石原石】：采玉坊及河床，可赌石开出翡翠。
    2. 【翡翠/玛瑙】：高级玉石，极高交易价值。
    3. 【盐】：海边或商船，烹饪及任务物品。
    4. 【沙石】：建筑材料，偶尔用于暗器。

## 3. 大研镇区域 (Dayan Town)
- **特点**：极其繁华，建筑错落，势力盘根错节。
- **盛产资源**：
    1. 【精金/镔铁】：铁匠铺及宝库，高级武器核心材料。
    2. 【丝绸/锦缎】：布庄及木府，制作高级护甲。
    3. 【陈年佳酿】：酒楼及俏梦阁，用于交友及回气。
    4. 【各类书信】：通过偷窃或搜索获得，开启隐藏任务。
    5. 【古玩珍宝】：珍宝阁及富户家，换取大量名望或金钱。

## 4. 遂宁与丈人村区域 (Sunning)
- **特点**：丛林茂密，竹影横斜，猎人居所。
- **盛产资源**：
    1. 【翠竹】：竹林采集，用于制作长柄武器及陷阱。
    2. 【兽皮】：狩猎野兔、野猪获得，制作皮甲。
    3. 【山珍/蘑菇】：雨后林间，珍稀食材。
    4. 【金银花/甘草】：药田盛产，解毒良药。

## 5. 卧龙村与石家庄区域 (Wolong)
- **特点**：依山而建，梯田层叠，石堡坚固。
- **盛产资源**：
    1. 【稻米】：梯田特产，主食资源。
    2. 【寒铁】：后山深处矿脉，打造神兵必备。
    3. 【花种】：庄园花园，陶冶情操。
    4. 【石块】：随处可见，投掷类暗器基础。

## 6. 极地与秘境 (Forbidden Realms)
- **极北雪域**：
    - 特点：终年积雪，生灵罕至。
    - 资源：【天山雪莲】、【白熊胆】、【万年冰泉】。
    - 🔮 **河图洛书**：【洛书·天璇碎片】藏于冰峰之巅，需抗寒装备+轻功≥20
- **西南密林**：
    - 特点：瘴气弥漫，蛇虫鼠蚁。
    - 资源：【五毒（蝎子、蜈蚣等）】、【断肠草】、【沉香木】。
    - 🔮 **河图洛书**：【洛书·天玑碎片】藏于毒瘴深处，需持有【避毒珠】
- **月牙岛**：
    - 特点：孤悬海外，地火涌动。
    - 资源：【玄铁】、【赤火砂】、【深海贝壳】。
    - 🔮 **河图洛书**：【河图·离位残片】藏于火山口，需抗火装备或内功≥25

## 7. 圣堂石窟 (Sanctuary Cavern)
- **特点**：远古圣堂遗址，蕴含天外意志，觉醒之地。
- **资源**：【圣堂断碑】、【正气歌诀】。
- 🔮 **河图洛书**：【河图·乾位残片】藏于深处，共鸣强度≥15时显现
