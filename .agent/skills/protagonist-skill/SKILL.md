# 主角核心能力 (Protagonist Skill)

本 Skill 用于管理和驱动武侠角色扮演游戏中的主角。

## 核心职责

1. **状态维护**：读取和更新主角的属性（HP, MP, 力道, 身法, 内功等）。参考 `references/character_sheet.md`。
2. **武学逻辑**：在战斗或交互中，根据主角已掌握的武学（如：太极拳、独孤九剑）计算效果。
3. **剧情输出**：结合主角的背景、性格和当前环境，产出高质量的武侠文学描写。

## 使用流程

1. **载入数据**：每轮对话开始，首先查看 `references/character_sheet.md` 确认主角当前的生存状况和能力值。
2. **接收意图**：解析用户的自然语言输入（如：“我决定在这间客栈打听消息”）。
3. **判定与反馈**：
   - 如果用户执行动作，判断是否需要属性判定（如：轻功是否足够翻过围墙）。
   - 结合环境 NPC 的 Skills（`npc-skill`），输出一段跌宕起伏的武侠故事情节。
4. **更新状态**：如果情节导致主角受伤、获得物品或习得新武功，及时更新 `references/character_sheet.md`。

## 武侠风格指南

- **用词考究**：使用江湖术语（如：寒暄、切磋、走火入魔、登峰造极）。
- **动静结合**：描写招式时，注重招式名称、劲力变化及环境影响。
- **快意恩仇**：剧情应体现武侠世界中的道义、背叛和宿命感。
