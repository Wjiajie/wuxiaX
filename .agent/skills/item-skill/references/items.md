# 物品、装备与弹药库 (Item Encyclopedia)

## 武器类别 (Weapon Types)
1. **剑 (Swords)**: 蛇形剑、松风剑、乌金剑、**玄铁剑**。
2. **刀 (Blades)**: 朴刀、砍柴刀、冷艳锯、**毒血刀**。
3. **长柄 (Polearms)**: 竹竿、红缨枪、关刀、**盘龙棍**。
4. **短兵 (Short Arms)**: 铁扇、匕首、**盘古菜刀**。
5. **暗器 (Hidden Weapons)**: 飞刀、铁蒺藜、**寻欢刀**。
6. **弓箭 (Bows)**: 桑木弓、铁胎弓、**后羿弓**。

## 消耗品与弹药 (Consumables)
- **恢复类**：槟榔（回气）、小还丹（回血）、大还丹（根治内伤）。
- **弹药类**：铁箭、精钢箭、流火矢（附带燃烧）、毒箭。
- **投掷类**：霹雳弹（大范围爆炸）、石子、**雷神之锤**（特殊投掷物）。

## 锻造材料 (Materials)
- **金属**：铜、铁、镔铁、寒铁、玄铁。
- **杂项**：雪豹筋、紫颤木、兽皮、天山雪莲。
